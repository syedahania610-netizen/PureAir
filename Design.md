Design

System Overview
PureAir follows a simple client server architecture with AI integration.

Architecture Flow
User
Frontend application
Backend server
AQI data source
AI recommendation engine
Personalized output

Frontend Design
Displays AQI values
Shows health recommendations
Collects user health details
Provides chatbot interface

Backend Design
Handles user requests
Fetches AQI data
Processes health profiles
Connects with AI engine

AI Recommendation Logic
Uses AQI level
Uses user health condition
Generates safety advice

Sample Recommendation
Good AQI safe to go outside
Moderate AQI limit outdoor activity
Poor AQI avoid going outside
Severe AQI stay indoors

Security Design
Uses secure communication
Stores minimal user data

Scalability
Supports multiple users
Can be deployed on cloud

Future Improvements
Multi language support
